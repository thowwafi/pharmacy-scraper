# pharmacy-scraper

```
pip install -r requirements
cd src
python scraper.py --search-qty 2 --max-pages 100 --continue-scraper

options
--search-qty: how many google search suggestions need that to be scraped
--continue-scraper: use this args to continuing data or start over
--max-pages: maximum page to be scraped

--start-index: start scraper on specific index 
--end-index: end scraper on specific index 
```
